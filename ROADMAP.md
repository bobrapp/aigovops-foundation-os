## Agent Suite Build Order

### Wave 1 (Weeks 1-6)
- Scribe/Capture Agent (voice to GDR draft)
- Triage/Synthesis Agent (dedupe, route, summarize)
- Mapper Agent (NIST/ISO/EU linkouts)
- Release Agent (versioning, changelogs, docs)
- Safety/Redaction Agent (consent + PII)

### Wave 2 (Months 2-6)
- Policy Compiler Agent (GDR to OPA/Rego/Sentinel/YAML)
- Dissent Amplifier Agent (detect minority concerns)
- Translation/Localization Agent (top 5 languages)

### Wave 3 (Months 6-18)
- Certification Ops Agent
- Grant Scout/Writer Agents
- Partner/CRM Agent
